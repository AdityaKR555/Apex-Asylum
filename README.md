# 🏋️ Apex Asylum Gym Website
 
A fully responsive gym landing page built with pure **HTML**, **CSS**, and **vanilla JavaScript**. Designed to showcase membership plans, services, a BMI calculator, and a contact/join form — all with a bold dark aesthetic.
 
---
 
 
## 🚀 Features
 
- **Sticky Navbar** with scroll blur effect and expand animation on page load
- **Hero Section** with a full-screen background and CTA button
- **About Section** with feature cards and an auto-scrolling image gallery
- **Services Section** with a built-in **BMI Calculator**
- **Membership Plans** — Basic, Standard, and Premium with highlighted active plan
- **Contact Section** with a join form and embedded Google Maps
- **Success Popup** on form submission
- **Fully Responsive** — mobile-friendly layout at `≤768px`
- **Smooth Scroll** navigation
 
---
 
## 🛠️ Tech Stack
 
| Technology | Usage |
|------------|-------|
| HTML5 | Structure & semantic markup |
| CSS3 | Styling, animations, media queries |
| JavaScript (Vanilla) | BMI logic, form handling, popups, scroll effects |
| Google Fonts | Fjalla One typeface |
| Google Maps Embed | Location iframe in contact section |
